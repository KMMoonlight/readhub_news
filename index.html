微软计划全面改革 GitHub，以抗衡 AI 编程工具竞品并布局 AI 智能体领域

微软计划对 2018 年收购的全球领先软件开发协作平台 GitHub 进行全面改革，以抗衡 AI 代码开发竞品，布局人工智能智能体领域，目前已启动团队重组。2025 年 1 月，微软组建「核心人工智能平台与工具部」，整合相关力量，虽整合后微软与 GitHub 仍保持一定程度独立运营，但微软一直在进行人员与资源调配。本周最新调整是将一个小型工程师团队划归 GitHub。此次调整目标一是抗衡竞争的 AI 代码工具，二是在人工智能智能体研发赛道抢占先机。微软希望 GitHub 的 AI 工具覆盖开发者全场景，将其打造成一站式操作平台。此外，调整也是 GitHub 基础功能升级计划的配套举措，升级重点包括优化自动化工具、开发数据分析与洞察工具、强化安全防护体系、推动平台业务在更多国家落地。


清华大学研发 AI 药物筛选平台，速度提升百万倍，全球开放数据库

清华大学智能产业研究院兰艳艳教授联合团队成功研发 AI 驱动的超高通量药物虚拟筛选平台 DrugCLIP，成果发表于《科学》杂志。该平台解决了靶向药物研发中在广阔化学空间快速筛选有效化合物的难题，筛选速度较传统方法提升百万倍，且预测准确性显著进步。研究团队依托该平台首次完成覆盖人类基因组规模的药物虚拟筛选，涉及约 1 万个蛋白靶点等，分析筛选超 5 亿个类药小分子，富集出超 200 万个潜在活性分子。团队构建的蛋白 - 配体筛选数据库已成为全球最大规模且向全球科研社区免费开放。
